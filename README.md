# tab-to-minichord
tab or note to minichord

tab 악보 혹은 음을 입력하면 가장 최적화된 미니코드 신디사이저 악보를 만들어 주는 프로그램.
https://minichord.com
<img width="644" height="254" alt="image" src="https://github.com/user-attachments/assets/6f445027-f649-43c2-90a0-4c1256173cec" />


입력 형식1: 기타 줄 번호1, 프랫 번호1 기타 줄 번호n, 프랫 번호n ... ...
입력 형식2: C4 D5 E6 . . .

음역대 제한: C3 - B7
